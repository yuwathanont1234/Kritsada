# UI Discover → Import (AI Solution)

ความสวยตัดสินที่ "reference + design spec" ไม่ใช่เครื่องมือ — เริ่มจากหา reference เสมอ

## จังหวะ 1 — DISCOVER (เริ่มที่นี่ทุกครั้ง)
1. ค้นหาแอประดับโลก 3 ตัวในหมวดเดียวกับแอปเป้าหมาย (ผ่าน web search / Mobbin)
2. แสดงภาพประกอบจริงของแต่ละตัว พร้อมระบุ:
   - จุดเด่น UX, รูปแบบ navigation
   - สิ่งที่ขโมยมาใช้ได้
   - สิ่งที่ต้องปรับให้เข้ากับผู้ใช้ไทย
3. **เสนอเป็น 3 choice ให้เจ้าของเลือกก่อน — หยุดรอ อย่าเพิ่งออกแบบหน้าจอ**

## จังหวะ 2 — IMPORT (หลังเลือก 1 ทิศทาง)
- ถ้ามีลิงก์ Figma Community → ใช้ Figma MCP ดึง component/สี/typography จริง
  (หมายเหตุ: Figma MCP อ่านได้เฉพาะไฟล์ที่ชี้ลิงก์ให้ ไม่ได้ท่องคลังทั้งโลก)
- ถ้าไม่มีลิงก์ → ร่าง design spec เองตามทิศทางที่เลือก

## หลักการออกแบบ
- mobile-first, ภาษาไทยเป็นหลัก, เรียบง่ายแต่ทรงพลัง
- ระบุหน้าจอหลัก 3-5 หน้า + flow การใช้งาน
- ส่งต่อ design spec ให้ ake-software-architect
